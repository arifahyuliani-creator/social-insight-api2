# Social Media Insight & Network API Pipeline

Proyek ini adalah sistem Back-End berbasis **Python (FastAPI)** dan **PostgreSQL** yang dirancang untuk kebutuhan:
1. **API Ingestion & Crawling:** Menarik data teks otomatis secara real-time.
2. **AI/NLP Processing Pipeline:** Analisis sentimen otomatis menggunakan Hugging Face Transformers.
3. **Social Network Analysis (SNA):** Mengukur metrik pengaruh user dalam jaringan menggunakan NetworkX.
